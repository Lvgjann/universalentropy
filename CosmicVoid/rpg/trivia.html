Roleplay Games

Overview

I created a multiverse RPG using four universes : Bioshock , Bioshock Infinite , Mad Max and Joueur du Grenier's RPG serie Les Suvivants. The main universe is based on the post apocalyptic zombie world of Les Survivants and I kept the same game system as Joueur du Grenier. The three other universes use an easier game system as they have been used for one-player sessions.
